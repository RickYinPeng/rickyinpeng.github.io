三.引用传递呢？

在Java中引用传递其实也是值传递，只是这个引用它本身保存的是此引用指向的内存地址，所以会将这个内存地址的值传过去，而我们在函数中操作此临时变量就相当于操作原来的引用，因为此时你是通过地址来操作啊，就是c语言中的指针
